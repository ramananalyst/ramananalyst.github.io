A Little Bit of SQL

March 1, 2025

Hey there! I’ve just started with SQL, and it’s okay for simple data stuff. I’m way more into C++ and JavaScript, but SQL is useful sometimes.

My SQL Book Viewer is a web app where you can search for books. I used basic SQL commands to find book titles and linked it to a webpage with JavaScript. I messed up some queries at first, but practicing helped.

SQL is good for quick data searches, but I prefer C++ for heavy tasks. My tip: try SQL with a small database and use MySQL Workbench. It’s not my main thing, but it’s nice to know a bit!
